About The Museum

The Roua De Jack Khoff Museum is a fictional museum created as part of a university web technologies project.

The museum focuses on digital exhibitions inspired by modern art, architecture and immersive visual storytelling.

This website was developed using HTML, CSS and Bootstrap with responsive design techniques.
